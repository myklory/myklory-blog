#使用github和hexo配置静态博客
###**1. 注册github和生成博客空间**
略
###**2. 安装Nodejs和hexo**
####**2.1 安装Nodejs**
[Node.JS](www.nodejs.org)
####**2.2 安装Hexo**
NodeJS安装好后使用下面命令安装Hexo:
```bash
npm install -g hexo
```
新建一个博客存储目录，进入目录后执行以下命令:
```bash
hexo init
npm install
```
